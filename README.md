📊 Excel AI Agent (CLI Tool)

A lightweight Python-based command-line tool that allows users to interact with Excel files using simple natural-language-like queries. The agent performs fast data analysis such as sums, averages, filtering, searching, and top-value extraction without requiring Excel formulas or advanced coding knowledge.

🚀 Key Features
Load and read any .xlsx file
Perform quick data analysis:
Sum, Average, Max, Min, Count
Query data using simple commands like:
total sales
average marks
max salary
Filter rows using conditions:
sales where region = west
Search across all columns automatically
Lightweight and easy to run (no UI required)
⚙️ Tech Stack
Python
Pandas
Openpyxl
🎯 Purpose

This project demonstrates how Excel data can be turned into an interactive query-based system using Python, making data analysis faster, simpler, and more accessible without writing complex formulas.
